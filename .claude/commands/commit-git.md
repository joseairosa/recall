# commit-git

Commit and push the current changes to the remote repository.

## Requirements

1. **Commit Message**: Write a conventional commit message (type: subject). No attribution to Claude, AI, or any assistant.

2. **Author/Committer**: Use the git user already configured in the local repository. Do not modify git config or add co-authored-by trailers.

3. **Forbidden Patterns** - Never include:
   - "Created by Claude"
   - "Generated by AI"
   - "Co-authored-by: Claude"
   - Any AI/assistant attribution

4. **Execution**:

```bash
   git add -A
   git commit -m "<type>: <concise description>"
   git push origin HEAD
```

5. **Commit Types**: feat, fix, refactor, docs, test, chore, perf, ci, build

## Example

```bash
git add -A
git commit -m "feat: add workspace isolation for hybrid mode"
git push origin HEAD
```
